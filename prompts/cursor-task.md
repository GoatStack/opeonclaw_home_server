# Cursor Task Prompt Template

아래 형식으로 Cursor에 작업을 요청한다.

---

## 목표
작업 목표를 한 문장으로 적는다.

## 요구사항
- 필요한 기능 1
- 필요한 기능 2
- 필요한 기능 3

## 제약사항
- 기존 구조 최대한 유지
- 불필요한 라이브러리 추가 금지
- 변경 이유를 코드 주석 또는 커밋 메시지로 남기기

## 출력 형식
- 변경한 파일 목록
- 핵심 변경 내용 요약
- 남은 TODO

## 테스트
- 가능한 경우 실행 방법 제시
- 실패 시 원인 요약

---

예시:

목표: tasks/inbox.md의 첫 번째 항목을 구현하기 위한 초기 스크립트 생성
요구사항:
- scripts/run_task.sh 생성
- tasks/ 폴더에서 첫 작업 파일 읽기
- logs/에 실행 로그 남기기
